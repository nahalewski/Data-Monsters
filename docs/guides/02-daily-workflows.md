# 2) Daily Workflows

## Morning Workflow

1. Open your dashboard and review priority tasks.
2. Check overdue activities and reschedule as needed.
3. Review new leads assigned overnight.
4. Confirm calendar sync and upcoming meetings.

## Lead Follow-Up Workflow

1. Open a lead record.
2. Review source, status, and last activity.
3. Add a call/email note.
4. Update lead status (new, working, qualified, disqualified).
5. Create a next-step task with due date.

## Meeting Follow-Up Workflow

1. Open the related contact/account.
2. Add meeting notes in the activity timeline.
3. Create action items and assign owners.
4. Update opportunity stage if required.
5. Set reminders for follow-up.

## End-of-Day Workflow

1. Mark completed tasks.
2. Update opportunity amounts/stages.
3. Ensure next actions exist for active pipeline items.
4. Review tomorrow's scheduled tasks.

## Productivity Tips

- Use filters and saved views to reduce context switching.
- Use keyboard shortcuts where available.
- Batch similar updates in one session.
