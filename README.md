# Data-Monsters

Persona Hub AI is a simple website for hosting multiple AI chat personas so users can pick a personality and chat in separate threads.

## Features

- Multiple built-in AI personas (mentor, coder, strategist, etc.)
- Custom persona creation with your own system prompt
- Separate chat history per persona
- OpenAI API integration (Responses API)
- Demo mode that works without an API key
- Light/dark theme toggle

## Run locally

Because this is a static app, you can run it with any simple web server:

```bash
python3 -m http.server 8000
```

Then open:

- http://localhost:8000
