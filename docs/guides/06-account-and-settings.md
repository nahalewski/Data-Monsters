# 6) Account & Settings

## Profile Settings

- Update name, title, and profile image.
- Set your language and timezone.
- Confirm preferred date/time format.

## Notification Settings

- Email notifications for assignments and mentions.
- In-app alerts for task due dates and pipeline changes.
- Optional digest summaries.

## Personal Preferences

- Default home dashboard.
- Default record list views.
- Signature and communication defaults.

## Security Basics

- Use strong, unique passwords.
- Enable multi-factor authentication if available.
- Do not share credentials.

## Session and Access

- Sign out from shared devices.
- Review active sessions periodically.
- Contact your administrator for role/access changes.
