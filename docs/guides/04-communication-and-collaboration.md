# 4) Communication & Collaboration

## Logging Communication

- Add call summaries with date/time and outcome.
- Save email context in activity timeline.
- Tag teammates when collaboration is needed.

## Internal Notes vs Customer-Facing Content

- **Internal notes**: For team context and planning.
- **Customer-facing fields**: For data shared in templates or portals.

## Assigning Work

1. Create a task from the record timeline.
2. Select assignee and due date.
3. Add clear action description.
4. Track completion in activity feed.

## Using Mentions and Alerts

- Mention users with @name in comments.
- Use alerts for high-priority accounts.
- Avoid unnecessary team-wide notifications.

## Communication Standards

- Keep notes concise and action-oriented.
- Include next step and owner.
- Avoid sensitive data in free-text fields unless policy allows it.
