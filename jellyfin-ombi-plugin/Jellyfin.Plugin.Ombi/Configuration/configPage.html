Ombi Requests

Connect Jellyfin to your Ombi server so users can search for and request movies and TV shows. Users open the request page from Dashboard → My Plugins → Ombi Requests or via the direct URL shown below.

The full base URL of your Ombi server, including scheme and port.
Found in Ombi under Settings → Configuration → General.
When enabled, requests are submitted using the Jellyfin username so they appear under the matching Ombi account. Requires the usernames to match in Ombi.
When enabled, requesting a show asks Ombi for every season unless the user chooses otherwise.
