# CRM Tool End-User Guides

Welcome to the CRM Tool end-user documentation hub. This section is organized by subject so you can quickly find the instructions you need.

## How to Use This Guide Set

- Start with **Getting Started** if you're new to the system.
- Use the **Daily Workflows** section for common day-to-day tasks.
- Use **Data & Records** for managing contacts, accounts, and opportunities.
- Use **Communication & Collaboration** for email, notes, and team coordination.
- Use **Reporting & Insights** to understand dashboards and exports.
- Use **Account & Settings** for profile preferences and notifications.
- Use **Troubleshooting & FAQ** when something isn't working as expected.
