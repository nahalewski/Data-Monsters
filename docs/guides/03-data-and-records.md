# 3) Data & Records Management

## Creating a New Contact

1. Go to **Contacts**.
2. Select **Create Contact**.
3. Enter required fields (name, email, phone, company).
4. Link to an existing account or create one.
5. Save and log initial interaction.

## Updating Accounts

- Keep company details current (industry, size, location).
- Maintain ownership and account status.
- Track relationship health with notes and tags.

## Managing Opportunities

1. Create a new opportunity from account or contact page.
2. Set expected close date and estimated value.
3. Assign stage (qualification, proposal, negotiation, closed won/lost).
4. Add stakeholders and activity history.

## Data Hygiene Rules

- Avoid duplicate contacts/accounts.
- Use required fields and approved drop-down values.
- Standardize capitalization and abbreviations.
- Archive obsolete records instead of deleting, when possible.

## Import/Export Basics

- Use CSV templates for bulk imports.
- Validate column mappings before final upload.
- Export only the fields you need.

**Warning:** Bulk imports can create duplicates if identifiers are inconsistent.
