# 5) Reporting & Insights

## Dashboard Basics

- Use role-based dashboard views.
- Pin frequently used widgets.
- Refresh data before sharing metrics.

## Common Reports

- Pipeline by stage
- Win/loss trend
- Activity by rep
- Lead source performance
- Forecast vs target

## Building a Simple Report

1. Open **Reports**.
2. Choose object (leads, opportunities, accounts).
3. Add filters (date range, owner, status).
4. Select columns and grouping.
5. Save as personal or shared report.

## Sharing Reports

- Share links with team members.
- Schedule recurring report emails if enabled.
- Export CSV for offline analysis.

## Data Interpretation Tips

- Compare trends over consistent date ranges.
- Validate outliers before acting.
- Separate activity metrics from outcome metrics.
